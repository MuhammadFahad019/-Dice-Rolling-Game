# 🎲 Dice Rolling Game (C++)

📌 **Description**
---
This project is a simple console-based game developed in C++ that simulates rolling a standard six-sided dice. 
It utilizes the `<cstdlib>` and `<ctime>` libraries to generate completely random numbers, providing a realistic dice-rolling experience. 

The main focus of this project is to demonstrate the use of random number generation, memory seeding, and basic loop control structures rather than complex graphical interfaces.

⚙️ **Features**
---
* Simulates rolling a standard 6-sided dice.
* Generates completely random numbers between 1 and 6 on every roll.
* Interactive menu allowing the user to roll the dice multiple times without restarting the program.
* Clean, simple, and beginner-friendly command-line interface.
